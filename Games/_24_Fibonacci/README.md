# Fibonacci Streamlit App

A Streamlit-based interactive Fibonacci sequence calculator inspired by
the classic Al Sweigart implementation.

## Features
- Compute the **Nth Fibonacci number**.
- Automatic sequence preview for smaller N.

## How to Run
[DOne] **Fast Mode**: Iterative algorithm for high performance.
[Done] **Safe Mode**: Recursion with caching (LRU) for correctness.
