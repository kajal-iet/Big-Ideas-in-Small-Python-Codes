## ASCII DNA Animation — Streamlit Version

This project renders a scrolling ASCII DNA double-helix animation inside Streamlit.
It recreates the twisting ladder effect from the original terminal-based animation using HTML + JavaScript, ensuring smooth, non-blocking visuals.

# Features

Smooth, continuous DNA animation
Randomized nucleotide pairs (A–T, T–A, C–G, G–C)
Accurate ASCII double-helix structure
Runs independently inside Streamlit using lightweight JavaScript
Reusable run() function for easy embedding
