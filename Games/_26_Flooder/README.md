# üé® Flooder 

A web-based version of the classic **Flood-It puzzle game**, built with **Streamlit**.  
The goal: *Flood the entire board with a single color before you run out of moves.*

Originally inspired by Al Sweigart's version from *The Big Book of Small Python Projects*.

---

## üöÄ Features

- Flood fill algorithm (recursive)
- Multiple difficulty levels (Easy / Medium / Hard)
- Color mode + Color-blind accessible Shape mode
- Best Score Tracking (stored per session)
- Clustered board generation for fair gameplay
- Board rendered dynamically in Streamlit UI

---

## ‚ñ∂Ô∏è How to Play

1. The top-left tile is your starting point.
2. Click a color/shape to flood adjacent matching tiles.
3. Continue until the entire board is one color.
4. Win before your move counter hits zero!
