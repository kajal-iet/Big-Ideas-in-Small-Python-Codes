**Dice Math** is an interactive math quiz game where you add up the pips on randomly placed dice. It combines **ASCII-art dice** with a timed quiz to make practicing arithmetic fun.

### Features:
- Randomly generates **2â€“6 dice** on a canvas.
- Dice are **ASCII-art** and placed at random positions.
- **Timed quiz**: 30 seconds (configurable).
- **Points system**: Correct answers give points, incorrect answers deduct points.
- Interactive and visual way to practice **addition**.

### How to Play:
1. Start the game.
2. Observe the dice on the canvas.
3. Enter the **sum of all dice**.
4. Repeat until the timer ends.
5. Final score is calculated based on correct/incorrect answers.
