# ⏱ Seven-Segment Countdown Timer (Streamlit)

A retro seven-segment countdown timer using the original **sevseg.py** module from Al Sweigart’s book.

## ✅ Features
- Classic seven-segment ASCII digits
- Smooth countdown animation
- Hours/minutes/seconds formatting
- "BOOM!" alert at zero

## ✅ How to Use
1. Make sure `sevseg.py` is in the same folder as the app  
2. Enter the number of seconds  
3. Click **Start Countdown**  
4. Watch the seven-segment display update every second  

## ✅ Requirements
- Streamlit  
- sevseg.py (included in the same directory)
